### Mapping

The following table explains how the Oracle datatypes are mapped to the target databases.

| Oracle                | PostgreSQL                 | Snowflake       |
| :------------------- | :------------------------- | :-------------- |
| **Numbers**          |                            |                 |
| `TINYINT`            | `SMALLINT`                 | `TINYINT`       |
| `SMALLINT`           | `SMALLINT`                 | `NUMBER`        |
| `MEDIUMINT`          | `INT`                      | `NUMBER`        |
| `INT`                | `INT`                      | `INT`           |
| `BIGINT`             | `BIGINT`                   | `BIGINT`        |
| `DECIMAL`            | `NUMERIC`                  | `NUMBER`        |
| `FLOAT`              | `FLOAT`                    | `FLOAT`         |
| `DOUBLE`             | `DOUBLE PRECISION`         | `FLOAT`         |
| **Numbers Unsigned** |                            |                 |
| `TINYINT UNSIGNED`   | `SMALLINT`                 | `NUMBER`        |
| `SMALLINT UNSIGNED`  | `INT`                      | `NUMBER`        |
| `MEDIUMINT UNSIGNED` | `INT`                      | `NUMBER`        |
| `INT UNSIGNED`       | `BIGINT`                   | `NUMBER`        |
| `BIGINT UNSIGNED`    | `NUMERIC(21)`              | `NUMBER`        |
| `DECIMAL UNSIGNED`   | `NUMERIC(N+1)`             | `NUMBER`        |
| `FLOAT UNSIGNED`     | `DOUBLE PRECISION`         | `FLOAT`         |
| `DOUBLE UNSIGNED`    | `N/A`                      | `N/A`           |
| **Times**            |                            |                 |
| `DATE`               | `DATE`                     | `DATE`          |
| `TIME`               | `TIME`                     | `TIME`          |
| `DATETIME`           | `TIMESTAMP`                | `TIMESTAMP_NTZ` |
| `TIMESTAMP`          | `TIMESTAMP WITH TIME ZONE` | `TIMESTAMP_LTZ` |
| `YEAR`               | `SMALLINT`                 | `INT`           |
| **Strings**          |                            |                 |
| `CHAR`               | `CHAR`                     | `VARCHAR`       |
| `VARCHAR`            | `VARCHAR`                  | `VARCHAR`       |
| `TINYTEXT`           | `TEXT`                     | `VARCHAR`       |
| `MEDIUMTEXT`         | `TEXT`                     | `VARCHAR`       |
| `TEXT`               | `TEXT`                     | `VARCHAR`       |
| `LONGTEXT`           | `TEXT`                     | `VARCHAR`       |
| **Blobs**            |                            |                 |
| `BIT`                | `BIT`                      | `BINARY`        |
| `BINARY`             | `BYTEA`                    | `BINARY`        |
| `VARBINARY`          | `BYTEA`                    | `BINARY`        |
| `TINYBLOB`           | `BYTEA`                    | `BINARY`        |
| `MEDIUMBLOB`         | `BYTEA`                    | `BINARY`        |
| `BLOB`               | `BYTEA`                    | `BINARY`        |
| `JSON`               | `JSON`                     | `VARCHAR`       |
| **Sets**             |                            |                 |
| `ENUM`               | `VARCHAR(64)`              | `VARCHAR(64)`   |
| `SET`                | `VARCHAR(64)`              | `VARCHAR(64)`   |
